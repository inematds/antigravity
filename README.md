# AntiGravity — do Zero ao App Publicado com IA

Curso aberto do **[INEMA.CLUB](https://inema.club)** sobre como construir, publicar e
escalar **aplicações reais com IA** usando o **AntiGravity** (o editor de código com
agentes da Google). Sem virar programador: você descreve, os agentes constroem.

🔗 **Acesse o curso:** https://inematds.github.io/antigravity/

## O que tem

5 trilhas · 11 módulos · 77 tópicos — com exemplos, exercícios e **prompts prontos**.

1. **Fundamentos** — o que é o AntiGravity, no-code vs. apps com IA, setup do ambiente.
2. **Construir & Publicar** — do problema ao primeiro app no localhost, depois deploy seguro.
3. **Escalar a Produção** — sistema de clonagem e levar sites para GoHighLevel/WordPress.
4. **Profissionalizar** — fluxo de trabalho profissional e entrega com QA e segurança.
5. **Maestria & Biblioteca** — o framework CODA + a biblioteca de skills e prompts.

## Biblioteca de Skills & Prompts

Artefatos prontos para baixar e adaptar em [`skills/`](skills/): skill de clonagem,
conversão para GoHighLevel, Ultimate Debugger Prompt, configuração de MCPs, template
`claude.md` (CODA), checklist de QA Pre-Flight e uma coletânea de prompts.

## Como funciona

Site estático (HTML + Tailwind via CDN + JavaScript), sem build nem backend. O progresso
de leitura, as anotações e as preferências de tema ficam salvos no seu navegador.
